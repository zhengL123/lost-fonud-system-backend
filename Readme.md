# 项目经历

## 景区失物招领平台后端开发

### 项目概述
景区失物招领平台是一个为游客和景区工作人员提供失物信息发布、查询、认领和管理功能的系统。该平台解决了传统失物招领流程繁琐、信息不透明的问题，提高了失物找回率和用户满意度。

### 技术栈
- **后端框架**：Spring Boot 2.7.6
- **持久层**：MyBatis Plus 3.5.3.1、MySQL 8.0+
- **缓存**：Redis 5.0+
- **认证授权**：JWT (JSON Web Token)、Spring Security
- **API文档**：Springdoc OpenAPI
- **文件上传**：Commons FileUpload
- **工具库**：Lombok、Apache Commons Lang3
- **构建工具**：Maven 3.8+

### 主要功能模块

#### 1. 用户认证与授权模块
- 实现基于JWT的无状态认证机制
- 支持用户注册、登录、密码重置功能
- 基于角色的权限控制（RBAC）
- 实现自定义JWT过滤器，处理token验证和用户信息提取

#### 2. 失物信息管理模块
- 支持失物信息的发布、查询、修改和删除
- 实现失物信息的分页查询和条件筛选
- 支持多图片上传和管理
- 提供失物状态管理（丢失中、已找到、已归还）

#### 3. 认领记录管理模块
- 实现失物认领申请的提交和处理
- 支持认领记录的状态跟踪和管理
- 提供认领审核流程

#### 4. 公告管理模块
- 支持管理员发布系统公告
- 实现公告的查询和管理

#### 5. 热门搜索管理模块
- 实现热门地点和热门物品的统计和展示
- 提供搜索热词的管理功能

#### 6. 统计分析模块
- 实现失物类型分布统计
- 提供月度失物数量统计
- 支持热门地点分析

#### 7. 文件上传模块
- 实现图片文件的上传和管理
- 提供文件访问路径的生成和管理

#### 8. 消息管理模块
- 实现用户之间的消息通知功能
- 支持系统消息的发送和管理
